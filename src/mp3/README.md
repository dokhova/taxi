# Аудиофайлы для виджета релаксации

Эта папка содержит MP3 файлы для плейера релаксации:

- **pause_ru.mp3** - Медитация "Тишина в пути"
- **4444.mp3** - Дыхательная практика "Ровный темп"

## Инструкция по добавлению файлов

1. Загрузите файлы в эту папку через GitHub:
   - `pause_ru.mp3`
   - `4444.mp3`

2. После деплоя на Vercel файлы будут доступны по адресам:
   - `https://demo.spokspace.com/mp3/pause_ru.mp3`
   - `https://demo.spokspace.com/mp3/4444.mp3`

3. Плейер автоматически начнёт использовать эти файлы
